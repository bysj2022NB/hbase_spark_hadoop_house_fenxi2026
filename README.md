## 功能
分析订单完成率 Tables01
分析各类房间租金的最大值、最小值、平均值 Tables02
分析各类房间的受欢迎度(订房数/总数) Tables03 
分析用户偏好的租期：租期小于90天的为短期，大于90天为长期 Tables04
分析用户支付方式占比 Tables05
分析每个月的订单数变化(状态：已完成) Tables06
不同面积区间租房数量 Tables07
